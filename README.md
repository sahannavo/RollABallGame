# 🎮 Roll-a-Ball: Overdrive Edition

A classic 3D arcade rolling game built using the **Unity Engine**. This project serves as a foundational exploration into Unity's physics engine, player input handling, dynamic camera scripting, and user interface systems.

## 🕹️ Gameplay & Features

* **Physics-Driven Controls:** Smooth, momentum-based ball movement utilizing Unity's `Rigidbody` physics.
* **Smart Camera Tracking:** A custom tracking script that maintains a constant offset, preventing dizzying camera rotations as the ball turns.
* **Dynamic Collection System:** Interactive collectibles that utilize trigger volume tracking to update player progression and live score counts.
* **Win Conditions:** Responsive UI system that tracks real-time scores and updates the game loop instantly upon completion.
